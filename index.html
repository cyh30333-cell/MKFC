실시간 공유 설정구글 시트 연결
연결하는 법 (5분, 무료)
  1. 구글에서 새 스프레드시트 하나 만들기 (sheets.new)
  2. 상단 메뉴 확장 프로그램 → Apps Script 열기
  3. 기존 코드 지우고, 아래 코드를 전체 복사해 붙여넣기
  4. 상단 배포 → 새 배포 → 유형: 웹 앱
  5. 액세스 권한: 모든 사용자로 설정 후 배포 (권한 승인 필요)
  6. 나오는 웹앱 URL(…/exec)을 복사해 위 칸에 붙여넣고 연결
  7. 팀원 배포 시엔 이 URL을 파일 상단 API_URL에 넣어 공유하면 모두 자동 연결
